# agro-recycler


Steps to Deploy in Local: 
  1. Clone the Repository
  2. cd server
  3. npm install
  4. npm start
  5. cd ..
  6. cd src
  7. npm install
  8. npm start



Technoclogy Stack Used :

  1. UI - React
  2. External API - IBM Watson Dicovery
  3. Server - Express


Reference: https://github.com/watson-developer-cloud/node-sdk
